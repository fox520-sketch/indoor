# 室內導航工具箱（改良版）

這是可直接上傳 GitHub 的單檔前端版本。

## 本次已實作

- GPS 柔性位置校正
- 地圖縮放、拖曳、雙擊縮放
- 指北針與縮放比例顯示
- 在地圖上直接新增標定點
- 以 GPS 建立標定點
- 以標定點校正目前位置
- GPS + 步頻自動估算步長（平滑更新）

## 使用方式

直接開啟 `index.html` 即可。
建議使用手機瀏覽器測試感測器、方向與定位能力。

## 注意

- 目前 QR 產生仍使用外部服務產圖。
- GPS 在室內精度受環境影響很大，因此本版採用柔性融合，不做硬跳點。
- 這是前端原型，資料儲存在 localStorage。
